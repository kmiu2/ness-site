# The NESS Site

## Getting Started

Run the site

```bash
$ npm i
$ npm run start
```

## Deployment

### SSH

SSH in to get started. (Replace with your WatIAM)

```bash
$ ssh kmiu@corn-syrup.csclub.uwaterloo.ca
# Optional if you have proper permissions set up
$ become_club ness
$ cd ../ness/
```

Run build

```bash
# Logged in and navigated to /users/ness/
$ git pull
$ npm run build
```
